# my little parser -- tooling is magic

to use demos:

```bash
$ npm install
$ cd demos
$ cd 00-ast
$ npm start # opens a browser; click to progress
$ # all other demo folders just contain "command.sh" 
# which assumes `node_modules/.bin` is on $PATH 
```
